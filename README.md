# MLB Predictions

Daily MLB game predictions powered by a stacked ensemble model combining XGBoost, LightGBM, and ridge regression.

## What's Here

This repo hosts the daily HTML prediction report generated by the [MLB-Model](https://github.com/I-R0N/MLB-Model) pipeline (private).

Each report includes:
- Win probability predictions for every scheduled MLB game
- Comparison against Polymarket betting odds
- Identified betting edges where the model disagrees with the market
- Model confidence levels and feature importance breakdowns

## How It Works

The prediction pipeline runs daily and:
1. Collects latest game data, player stats, and Statcast metrics
2. Engineers features (Elo ratings, rolling stats, matchup context, park factors)
3. Generates predictions via stacked ensemble
4. Compares against Polymarket odds to find edges
5. Publishes the report here
